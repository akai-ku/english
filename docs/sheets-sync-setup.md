# Googleスプレッドシート同期のセットアップ手順

英文カードのデータをGoogleスプレッドシートに保存できるようにする手順です。
初回だけ10分ほどの作業が必要です。スマホでもできますが、PCのほうが楽です。

## 仕組み

スプレッドシートに「Apps Script」(Googleの無料スクリプト機能)で小さな窓口を作り、
英文暗記アプリからそこへ保存・読み込みします。

- **ページを開いたとき** … 自動でスプレッドシートの最新データを読み込みます
  (この端末に未保存の変更があるときはスキップして知らせます)
- **💾 保存** … アプリのデータでシートを丸ごと上書き
- **⚙️ → 📥 いますぐ読み込み直す** … 手動でシートのデータを読み込み(確認あり)

複数の端末で使う場合は「使い終わったら💾保存」だけ守れば、
次にどの端末で開いても最新のデータが自動で入ります。

## 手順

### 1. スプレッドシートを作る

1. https://sheets.new を開いて新しいスプレッドシートを作成
2. 名前を付ける(例:「英文暗記カード」)

### 2. Apps Scriptを設置する

1. メニューの **拡張機能 → Apps Script** を開く
2. エディタに最初から入っているコードを全部消す
3. このリポジトリの [`gas/Code.gs`](../gas/Code.gs) の内容を全部コピーして貼り付ける
4. 上のほうにある `SECRET_KEY = "ここを自分の合言葉に変える"` を、
   自分だけの合言葉に書き換える(例: `"english-yamada-2026"`)。
   **推測されにくい文字列にしてください**
5. 💾(保存)を押す

### 3. ウェブアプリとしてデプロイする

1. 右上の **デプロイ → 新しいデプロイ** を押す
2. 左上の歯車アイコン → 種類で **ウェブアプリ** を選ぶ
3. 次のように設定する:
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: **全員**
4. **デプロイ** を押す
5. 「アクセスを承認」を求められたら:
   1. 自分のGoogleアカウントを選ぶ
   2. 「このアプリはGoogleで確認されていません」と出たら **詳細** →
      **(プロジェクト名)に移動** を押す(自分で作ったスクリプトなので安全です)
   3. **許可** を押す
6. 表示された **ウェブアプリのURL**(`https://script.google.com/macros/s/…/exec`)をコピーする

### 4. 英文暗記アプリに設定する

1. 英文暗記アプリを開いて **⚙️** を押す
2. **接続設定** を開いて、コピーしたURLと手順2-4で決めた合言葉を入力 → **設定を保存**
3. **💾 保存** を押す
4. 「✅ ◯枚をスプレッドシートに保存しました」と出て、
   スプレッドシートに `cards` シートができていれば完了です 🎉

## よくあるトラブル

| 症状 | 対処 |
| --- | --- |
| 「合言葉が一致しません」 | アプリの合言葉と `Code.gs` の `SECRET_KEY` が同じか確認 |
| 「応答を読み取れませんでした」 | デプロイ設定の「アクセスできるユーザー」が **全員** になっているか、URLが `/exec` で終わっているか確認 |
| スクリプトを書き換えたのに反映されない | **デプロイ → デプロイを管理 → ✏️ → バージョン: 新バージョン → デプロイ** で更新(「新しいデプロイ」を作るとURLが変わってしまうので注意) |

## 注意事項

- **音声ファイル自体は同期されません**(サイズが大きいため端末内にのみ保存)。
  カードと音声クリップの割り当て位置は同期されるので、別の端末でも同じ音声を
  取り込んで同じ名前で使えば、区切りを付け直す必要はありません

- URLと合言葉の両方を知っている人はデータを読み書きできます。他人に教えないでください
- 「保存」も「読み込み」も**丸ごと上書き**です。読み込みはページを開いたときに
  自動で行われるので、**使い終わったら「💾 保存」を押す**習慣にするとデータが混ざりません
