# 🗣️ 英文暗記トレーニング

写真で撮った英文を暗記し、発音・和文英訳まで鍛えるためのシンプルな Web アプリです。
スラッシュリーディング教材(`A: Good morning, / Mr. Tanaka. // Can I ask you / …`
のように `/` `//` でチャンク区切りされた会話文)にそのまま対応しています。

ビルド不要・サーバー不要の静的ファイルのみ。データはブラウザの localStorage に保存されます。

## 学習の流れ(4段階メソッド)

「🎯 トレーニング」から、Input → Output を繰り返して英文を「わかる」から「使える」へ引き上げます。
上の段階から順に進めるのがおすすめです。

| 段階 | 種類 | トレーニング | ねらい |
| --- | --- | --- | --- |
| **①** | Input | 🗣️ シャドーイング / 🎯 オーバーラッピング / 📢 音読 / 🔍 精読 | 英文を体にしみ込ませる |
| **②** | Output | 🧠 リテンション / 🔁 リピーティング | ①をもとに発信力を強化(文法・語彙・表現の反復) |
| **③** | Input | ⚡ 頭ごなし訳(英→日) | ①〜②をもとに、発信のための強固な基盤づくり |
| **④** | Output | 💨 高速和文英訳 / ⚡ 頭ごなし訳(日→英) | ①〜③をもとに、伝えたいことを瞬時に英語で発信 |

各トレーニングの動き:

- **シャドーイング** … テキストを隠したまま音声を自動再生。少し遅れて“影のように”真似します(👁 で答え確認・🎤 で発音チェック)
- **オーバーラッピング** … テキストを見ながら音声に重ねて同時に発音
- **音読** … 声に出して読み、🎤 発音チェックで確認
- **精読** … チャンク対応表で文法・意味をていねいに理解
- **リテンション / リピーティング** … 音声だけを聞いて保持し、テキストを見ずに再現(🎤 で確認)
- **頭ごなし訳(英→日 / 日→英)** … `/` `//` の区切りを使い、語順のまま前からチャンクごとに訳を出していきます
- **高速和文英訳** … 日本語を見て、タイマーを計りながら素早く英語で発話 → 👁 で英文を確認

## 主な機能

- **📷 写真から英文を取り込み**
  - 英語ページを撮影すると [Tesseract.js](https://github.com/naptha/tesseract.js) がブラウザ内で OCR し、`A:`/`B:` の発話ごとに1枚のカードにします(1行 = 1カード。読み取り結果は登録前に編集可)
  - 日本語訳のページも撮影すると、行ごとに英文と対応づけて訳として登録できます
  - 訳がない行は [MyMemory API](https://mymemory.translated.net/) の自動翻訳で補えます(あとで編集可)
  - 写真が横向きのときは「↻ 回転」で直してから読み取れます
- **🎯 4段階トレーニング**(上記)
  - 各トレーニングは **🃏 カード練習**(1枚ずつページ送り)と **📖 全文練習**(会話全体をとおし)を切り替えられます
  - 全文練習では教材音声(または読み上げ)を全文連続再生でき、🎤 で全文の発音チェックもできます
  - 全文再生は区切りをつなぎ合わせるのではなく、時間が連続している区間は**元の音声をそのまま通しで再生**します(会話の自然な間合いが残ります)
  - 🎤 を開始するとお手本の再生は自動で止まります(マイクがお手本の音を拾うと採点にならないため)
- **🧠 暗記の復習(間隔反復)**
  - 「✗ もう一度 / ○ できた / ◎ 余裕」の自己採点で次の復習日を自動調整(翌日 → 3日 → 1週間 → …)
  - 「今日の復習」だけが出題されるので、毎日ひらくだけで続けられます
  - 日→英・英→日の両モード
- **🎧 教材音声の取り込み**
  - 教材付属の音声(mp3 / m4a など)を取り込むと、お手本の再生が機械音声のかわりに**本物の音声**になります
  - 音声はこの端末のブラウザ内(IndexedDB)にだけ保存され、どこにも送信されません
  - 「✂️ 区切りを付ける」で、再生しながら発話の切れ目をタップするだけでカードに割り当てられます
  - 割り当て後は、一覧・復習・全トレーニング(シャドーイングの自動再生含む)で教材音声が使われます(🐢 は 0.7倍速)
- **☁️ スプレッドシートへのバックアップ・同期**(任意)
  - ページを開くと自動で最新データを読み込み、「💾 保存」でスプレッドシートに保存
  - 端末の買い替えや複数端末での利用に対応。セットアップ手順は [docs/sheets-sync-setup.md](docs/sheets-sync-setup.md) を参照
  - 音声ファイル自体は同期されません(カードと割り当て位置のみ)
- **🔊 読み上げ / 🎤 発音チェック**(Web Speech API)
  - お手本の読み上げ(🐢 ゆっくり再生も)。教材音声が無いカードは TTS で読み上げ
  - マイクに話すと音声認識で聞き取り、単語ごとに一致・不一致を色分けして100点満点で採点

## 使い方

```bash
# 例: Python の簡易サーバーで起動
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

GitHub Pages などに置けばスマホからも使えます。

> **動作環境の注意**
> カメラ・マイク・音声認識は、セキュリティ上 `https://` または `http://localhost` でのみ動作します。
> 読み上げ・発音チェックはブラウザの Web Speech API を使うため、**Chrome / Safari** での利用をおすすめします。
> 写真の OCR と自動翻訳は外部への通信が発生します(登録時のみ)。
